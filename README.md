🖼️ تصویر کلی از Assetly

![داشبورد اصلی](https://raw.githubusercontent.com/AliDeveloop/asset-tracker-assetly/main/img/in1.png)


Assetly پلتفرمی جامع برای مدیریت حرفه‌ای سرمایه‌های شماست. چه در بازار ارز دیجیتال فعال باشید، چه طلا و سکه معامله کنید، یا ارزهای خارجی در اختیار داشته باشید، Assetly تمام دارایی‌های شما را در یک داشبورد قدرتمند جمع‌آوری و تحلیل می‌کند.

🛠️ آموزش نصب و راه‌اندازی
مرحله ۱: نصب پایتون
به سایت رسمی پایتون بروید:
🔗 https://www.python.org/downloads/

آخرین نسخه Python 3.x را دانلود کنید

هنگام نصب، حتماً "Add Python to PATH" را تیک بزنید:

نصب را کامل کنید

مرحله ۲: دانلود پروژه Assetly
دو روش برای دانلود:

روش ۱: از GitHub (توصیه شده)
cmd رو باز کنید 
# باز کردن Command Prompt یا PowerShell
# کپی کردن دستور زیر:

git clone https://github.com/AliDeveloop/asset-tracker-assetly.git

cd portfolio-tracker

روش ۲: دانلود مستقیم ZIP
از ریپازیتوری GitHub پروژه را دانلود کنید

فایل ZIP را در محل دلخواه Extract کنید

مرحله ۳: نصب کتابخانه‌های مورد نیاز
cmd رو باز کنید 
# در Command Prompt یا PowerShell، داخل پوشه پروژه:

pip install -r requirements.txt

اگر با خطا مواجه شدید:

cmd رو باز کنید 
# به صورت جداگانه نصب کنید:
pip install Flask

pip install requests

pip install beautifulsoup4

pip install apscheduler

مرحله ۴: اجرای برنامه
روش ۱: از طریق فایل portfoilio.bat
داخل پوشه پروژه، فایل portfoilio.bat را پیدا کنید

روی آن دو بار کلیک کنید

برنامه به صورت خودکار اجرا می‌شود

روش ۲: دستی در Command Prompt
cmd رو باز کنید 

python app.py
مرحله ۵: دسترسی به برنامه
مرورگر خود را باز کنید (Chrome, Firefox, Edge)

آدرس زیر را وارد کنید:


http://localhost:5000
Assetly برای شما بارگذاری می‌شود 🎉

🔧 عیب‌یابی مشکلات رایج
مشکل ۱: پایتون شناخته نمی‌شود
text
'python' is not recognized as an internal or external command
راه حل: پایتون را مجدداً نصب کنید و "Add to PATH" را تیک بزنید

مشکل ۲: خطای کتابخانه
text
ModuleNotFoundError: No module named 'flask'
راه حل: دستور نصب کتابخانه‌ها را مجدداً اجرا کنید

مشکل ۳: پورت ۵۰۰۰ در حال استفاده است
text
Address already in use
راه حل: برنامه‌های دیگر را ببندید یا پورت را تغییر دهید

مشکل ۴: فایل bat اجرا نمی‌شود
راه حل: روی فایل کلیک راست → Run as Administrator
